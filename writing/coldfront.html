Coldfront Installation Notes

Jan 10th, 2024

Intro

This is a small set of install notes for the resource allocation management system ColdFront. In this, I walk through the ColdFront Deployment docs, and address some pitfalls and hangups that I experience while setting the system up on a small HPC system for my University. Lastly I also describe configuration for the SAML SSO plugin I wrote for ColdFront, and how you should configure your IDP as far as what claims to send and how to format them for the plugin.
NOTE: These installation notes are pretty specific towards the HPC configuration they were made for: An OpenHPC cluster running their versions of Slurm on Rocky 8, with Microsoft EntreID as the IDP.
